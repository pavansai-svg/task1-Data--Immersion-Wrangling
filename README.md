##  Dataset Summary

- Customer information
- Product details

---

##  Data Issues Identified
- 20 missing values in *Price*
- 13 missing values in *City*
- 5 duplicate rows
- Inconsistent product name formatting
- Date columns not in proper format

---

##  Data Cleaning Steps Performed
- Removed duplicate records
- Replaced missing *Price* using median imputation
- Replaced missing *City* with "Unknown"
- Converted date columns to datetime format
- Standardized product names (lowercase)
- Created new derived features
